# 项目介绍

 	 该项目主要是通过chatgpt提供的python脚本，从各个电子书网站上收集的各类电子书下载链接. 

	 书籍根据常用的tag标签进行简单分类，点击对应的tag可以进入详细的下载链接地址，包括了epub、mobi、azw3三种格式文件。
